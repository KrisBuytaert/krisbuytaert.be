What machines use the PPC 403

A PPC 403 can be used in lots of different applications, probably the most known is the NC 100 you can already use the NC in a Linux Environment

More details on further products can be found at IBM's site where you can read all about it being supported by Winslows CE, so there a need for the Linux port too..
