(JMXTrans also comes with a Ganglia target option) </p> <p>Rather than going in depth over the config it's probably easier to point to a Vagrant box I build <a href="https://github.com/KrisBuytaert/vagrant-graphite" title="https://github.com/KrisBuytaert/vagrant-graphite">https://github.com/KrisBuytaert/vagrant-graphite</a> which brings up a machine that does pretty much all of this on localhost.</p> <p>Obviously it's still a work in progress and lots of classes will need to be parametrized and cleaned up.
