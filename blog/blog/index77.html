PhpWiki conversion 2 MediaWiki

By: Kris Buytaert

I found some interresting Documentation
on how to convert from our old PHPWiki to MediWiki in order to gain autenticated and email validated users.

Basically doing a zip dump from the PHPWiki , then running it to the mentionned sed script
only to redirect the content into the new MediaWiki database almost works perfectly.
Yes you have some minor tweaks to do for the external links, but those are acceptable.

Anyway .. that all to say that the old openMosix PHP Wiki is about to be upgraded to a new MediaWiki :)

Nov 14 2006

Sun relicenses Java under GPL

By: Kris Buytaert

In Sun relicenses Java under GPL Tom wonders who cares.

Not what we care that much, but it sure will be alot easier for a lot of distributinos to provide a correctly packaged JRE or JSDK. apt-get install java or yum install jre while not having to add non default enabled repositories , or even apt-get install jboss and get the whole dependency shebang in one pull surely beats having to go to java.sun.com then figure out under which section they have been hiding the JDK again, download it install it manually .

And just that little change is a good thing. :)

Inviting FOSDEM 2007 speakers
